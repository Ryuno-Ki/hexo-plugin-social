# hexo-plugin-social

[Hexo][hexo] plugin for widgets of social media sites.

## Usage

After installation of this Hexo plugin, you will be able to use new tags in
your posts:

```markdown
{% twitter "mytwitterhandle" %}
{% facebook "myfacebookhandle" %}
```
